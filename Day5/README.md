# Day 5 - Data Cleaning Tasks

## 📌 Objective
Handle missing values, remove duplicates, and understand dataset statistics.

## 📊 Topics Covered
- Import Pandas & NumPy
- Load Raw Dataset (with missing values & duplicates)
- Understand Dataset Statistics (`info()`, `describe()`, `isnull()`)
- Handle Missing Values (fill numeric columns with mean, fill text columns with placeholder)
- Detect & Remove Duplicate Rows
- Recalculate Derived Columns
- Save the Cleaned Dataset

## 📁 Files
- `Day5_Data_Cleaning.ipynb` – Jupyter Notebook containing all data cleaning tasks with code and outputs
- `student_scores_raw.csv` – Raw dataset (contains missing values & duplicate rows)
- `student_scores_clean.csv` – Cleaned dataset generated after running the notebook

## ✅ Expected Outcome
Clean dataset prepared.
