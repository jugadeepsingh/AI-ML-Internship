# Day 3 – NumPy Tasks

## Objective
Learn NumPy arrays, indexing, and mathematical operations. Practice array-based calculations.

## Topics Covered
- Creating NumPy arrays (1D, 2D, and using built-in functions like `zeros`, `ones`, `arange`, `linspace`, `random`)
- Array properties: shape, size, dimensions, data type
- Indexing and slicing (1D, 2D, boolean indexing)
- Mathematical operations: element-wise arithmetic, aggregate functions (sum, mean, max, min, std)
- Matrix operations: dot product, transpose
- Practice task combining array creation, slicing, filtering, aggregation, and normalization

NumPy fundamentals completed.
